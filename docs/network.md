# Network Topology

This file describes the network topology used in my homelab, including IP addressing and how the devices are connected.

## Subnets
- **LAN**: 192.168.1.0/24
- **DMZ**: 192.168.2.0/24

## Devices
- **Router**: 192.168.1.1
- **Homelab Server**: 192.168.1.10
- **NAS**: 192.168.1.20
- **Pi-hole**: 192.168.1.30
